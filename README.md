# AI Document Assistant (Offline)

An offline AI-powered document assistant built with:
- Flask
- LangChain
- Ollama (Local LLM)
- FAISS
- HuggingFace Embeddings

## Features
- Local LLM (no OpenAI key required)
- Document-based Q&A
- Streaming responses
- User login system
- Works fully offline

## Requirements
- Python 3.10+
- Ollama installed
- At least 4GB RAM
